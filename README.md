# Colaboratoryを利用したPython実行環境

## Colaboratory(以下Colab)の導入
https://colab.research.google.com/

## Colabの特徴
  - 環境構築が不要
  - GPU への無料アクセス

## Colabの設定
ツール-設定 から環境設定可能。  
パワーレベル、コーギーモード、猫モード、カニモードを必要に応じて設定すると良いでしょう。
